Project Overview

This project presents a comprehensive healthcare data analysis performed using Microsoft Excel. The objective is to transform raw healthcare data into meaningful insights through data cleaning, analysis, and interactive dashboards.

Features

Data cleaning and preprocessing
Interactive dashboard
Pivot Tables and Pivot Charts
KPI metrics and performance analysis
Patient demographic analysis
Treatment and diagnosis trends
Hospital and doctor performance insights
Excel formulas and conditional formatting
Slicers for dynamic filtering

Key Insights
Patient distribution by age and gender
Most common medical conditions
Treatment success trends
Hospital performance comparison
Admission and discharge analysis
Overall healthcare performance metrics
